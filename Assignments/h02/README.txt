## Definitions

#### Abstraction
>the process of removing physical, spatial, or temporal details or attributes in the study of objects or systems in order to focus attention on details of higher importance.


####Attributes / Properties
> an attribute is a specification that defines a property of an object, element, or file.


####Class
>Class are basic building blocks in object-oriented programming languages.


####Class Variable
>any variable declared with the static modifier of which a single copy exists, regardless of how many instances of the class exist.


####Composition
>the result of each function is passed as the argument of the next, and the result of the last one is the result of the whole.


####Encapsulation
>the bundling of data with the methods that operate on that data, or the restricting of direct access to some of an object's components. 


####Friends
>a function that is given access to the private and protected members of the class in which it is declared as a friend.


####Inheritance
>the mechanism of basing an object or class upon another object


####Instance Variable
>a variable defined in a class, for which each instantiated object of the class has a separate copy, or instance.


####Member Variable
>a variable that is associated with a specific object, and accessible for all its methods


####Method
>a procedure associated with a message and an object.


####Multiple Inheritance
>an object or class inherits characteristics and features from more than one parent object or parent class.


####Object
> a variable, a data structure, a function, or a method, and as such, is a value in memory referenced by an identifier.


####Overloading
> functions that can be applied to arguments of different types, but that behave differently depending on the type of the argument to which they are applied


####Polymorphism
>the provision of a single interface to entities of different types or the use of a single symbol to represent multiple different types.


####Public / Private / Protected
>keywords that set the accessibility of classes, methods, and other members.


####Virtual
>a digitally replicated version of something real.
